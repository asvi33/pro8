## 📌 Overview

The **NumPy Analyzer** is an interactive, menu-driven Python console application built entirely around the **NumPy** library. It lets a user create 1D, 2D, and 3D arrays, run mathematical operations on them, combine or split arrays, search/sort/filter values, and instantly compute key statistical aggregates — all through a simple text menu.

This project is designed to:
- Strengthen hands-on understanding of **NumPy array creation, indexing, and slicing**
- Practice **vectorized mathematical operations** (addition, subtraction, multiplication, division)
- Explore **array manipulation** techniques like stacking and splitting
- Apply **searching, sorting, and filtering** using NumPy's built-in functions
- Compute **descriptive statistics** — sum, mean, median, standard deviation, and variance

---

## 🎯 Problem Statement

> **Objective:** Build a console-based interactive tool that lets a user create NumPy arrays of any dimension and perform a full suite of analytical operations on them.

You are building a utility for students and practitioners learning NumPy. The program must accept user choices from a menu and execute the corresponding task — creating an array, performing arithmetic, combining/splitting, searching/sorting/filtering, or computing statistics — while validating input at every step.

| 📂 Feature | 📄 Type | 🔍 Description |
|------------|---------|----------------|
| Array Creation | Core | Builds 1D, 2D, or 3D arrays from user input |
| Indexing & Slicing | Core | Access single elements or sub-ranges of the array |
| Mathematical Operations | Logic | Element-wise add, subtract, multiply, divide with a second array |
| Combine / Split | Logic | Stack arrays vertically/horizontally or split them apart |
| Search, Sort, Filter | Logic | Locate values, sort in place, filter by threshold |
| Aggregates & Statistics | Logic | Sum, mean, median, standard deviation, and variance in one view |

The goal is to demonstrate **practical NumPy proficiency** through a clean, menu-driven interactive program.

## 🧩 Part A — Array Creation & Indexing

### 📝 1. What is Array Creation here?

The program builds a **NumPy array** from flat, space-separated user input and reshapes it into 1D, 2D, or 3D form, then immediately opens an **Indexing/Slicing** sub-menu for the newly created array.

### 🗺️ 2. Dimension Overview

| Dimension | Shape Logic | Example |
|-----------|-------------|---------|
| 1️⃣ 1D | Flat list of `n` elements | `[10 20 30]` |
| 2️⃣ 2D | `rows × cols` via `reshape(rows, cols)` | `[[10 20 30] [40 50 60]]` |
| 3️⃣ 3D | `x × y × z` via `reshape(x, y, z)` | Nested block array |

**Logic:**
```python
data = list(map(int, input(f"Enter {total} elements: ").split()))
self.array = np.array(data).reshape(rows, cols)
```

### 🎯 3. Indexing & Slicing

> Dimension-aware access — the program checks `self.array.ndim` and asks for the right number of coordinates.

**Logic:**
```python
if self.array.ndim == 2:
    r1, r2 = map(int, row.split(":"))
    c1, c2 = map(int, col.split(":"))
    print(self.array[r1:r2, c1:c2])
```

**Sample Output (2×3 array, rows 0:2, cols 1:3):**
```
[[20 30]
 [50 60]]
```

## ➗ Part B — Mathematical Operations

> Performs element-wise arithmetic between the created array and a second, user-supplied array of matching shape.

**Logic:**
```python
second = np.array(elements).reshape(self.array.shape)

if choice == "4":
    if np.any(second == 0):
        print("Division by zero is not allowed.")
    else:
        print(self.array / second)
```

**Key Concepts Used:**

| Concept | Detail |
|---------|--------|
| ➕ Element-wise Addition | `array + second` |
| ➖ Element-wise Subtraction | `array - second` |
| ✖️ Element-wise Multiplication | `array * second` |
| ➗ Safe Division | Checked with `np.any(second == 0)` before dividing |

## 🔍 Part D — Search, Sort & Filter

**Logic:**
```python
indices = np.where(self.array == val)          # Search
sorted_arr = np.sort(self.array, axis=1)        # Sort (row-wise for 2D)
filtered = self.array[self.array > threshold]   # Filter
```

| Concept | Function | Behavior |
|---------|----------|----------|
| 🔍 Search | `np.where()` | Returns indices where the value matches |
| ⬆️ Sort | `np.sort()` | Row-wise for 2D arrays, full sort otherwise |
| 🧮 Filter | Boolean Masking | Keeps elements greater than the threshold |

## 📊 Part E — Aggregates & Statistics

> Computes all five key statistics **together** in a single view, instead of one at a time.

**Logic:**
```python
print(f"Sum: {np.sum(self.array)}")
```
